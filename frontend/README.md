# NeuroStay AI

NeuroStay AI is a full-stack AI-powered hotel discovery platform built using React, TypeScript, Tailwind CSS, Node.js, Express, and MongoDB.

---

## Features

- AI-powered hotel search
- AI loading animation
- Dynamic hotel recommendations
- Match score system
- Hotel comparison
- Save hotel wishlist
- Login & Register authentication
- Protected routes
- Profile dashboard
- Search history
- Hotel detail page
- Google Maps integration
- Contact page
- About page
- 404 page
- Responsive modern UI

---

## Tech Stack

### Frontend
- React
- TypeScript
- Tailwind CSS
- React Router

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
